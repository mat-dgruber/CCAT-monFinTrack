# Orçamentos & Metas Mensais 🎯

Os **Orçamentos** permitem que você defina limites de gastos para cada categoria, transformando sua gestão financeira de reativa em proativa.

## 📋 Como Funciona

Para cada categoria de gasto (ex: Alimentação, Lazer), você define uma "Meta". O sistema monitora suas transações em tempo real e mostra quanto do orçamento já foi consumido.

## 🛠️ Configuração da Meta

### Alvo do Orçamento

- **Categoria de Gasto:** Selecione a categoria que deseja monitorar. Você só pode criar um orçamento por categoria principal.

### Definição de Limite

- **Limite Mensal Desejado:** O valor máximo que você pretende gastar naquela categoria no mês.
- **Alertas Inteligentes:** O sistema gera alertas visuais quando você atinge **80%** (Amarelo) e **100%** (Vermelho) do limite.

## 📊 Indicadores de Progresso

- **Barra de Progresso:** Mostra visualmente o preenchimento do orçamento.
- **Porcentagem:** Cálculo exato do uso. Se exceder 100%, a barra mudará para um estilo de "alerta de estouro".
- **Gasto vs. Meta:** Comparação direta em valores monetários para que você saiba exatamente quanto ainda pode gastar.

> [!TIP]
> Comece definindo orçamentos para suas maiores despesas variáveis, como Alimentação e Transporte, para ter um controle imediato de onde seu dinheiro está indo.
