# Gráficos Avançados & Drill-down 📊

Nossos widgets de gráficos não são apenas imagens estáticas; eles são ferramentas interativas de descoberta financeira.

## 📈 Tipos de Visualizações Especiais (Recurso Pro)

### Treemap (Mapa de Árvore)
Uma representação hierárquica por áreas. Quanto maior o retângulo, maior o gasto.
*   **Drill-down:** Clique em uma categoria principal para "entrar" nela e ver o detalhamento por subcategorias.
*   **Navegação:** Use os breadcrumbs (links no topo) para voltar ao nível anterior.

### Box Plot (Distribuição)
Ideal para entender a variabilidade dos seus gastos.
*   **Min/Max:** Os "bigodes" mostram o menor e maior gasto no período.
*   **Caixa (Q1-Q3):** Onde estão concentrados 50% dos seus gastos.
*   **Linha Interna:** A **mediana**. Se estiver muito acima ou abaixo do centro da caixa, indica uma tendência de gastos desequilibrados.

### Sankey (Fluxo de Caixa)
A visualização definitiva para ver de onde o dinheiro vem e para onde ele vai.
*   **Nós:** Representam fontes de renda, contas bancárias e categorias de saída.
*   **Links:** A largura das fitas representa o volume financeiro.
*   **Interação:** Use o **zoom (scroll)** e **arraste (drag)** para explorar fluxos complexos. Use o botão de reset para voltar à visão original.

## ⚙️ Configurações do Widget
Cada gráfico possui um ícone de engrenagem com opções poderosas:
*   **Comparar (Período Anterior):** Adiciona uma camada de dados do mês passado para ver se você melhorou ou piorou.
*   **Previsão (Linha de Tendência):** Calcula estatisticamente para onde seu saldo aponta no futuro.
*   **Resumo em Texto:** Transforma o gráfico em um painel de estatísticas puras (Total, Média, Maior Despesa).
*   **Exportar CSV:** Baixe os dados para usar no Excel ou ferramentas externas.

## 📐 Ajuste de Layout
Use o ícone de maximizar no widget para expandi-lo em duas colunas, ideal para análises detalhadas no Sankey ou Treemap.
