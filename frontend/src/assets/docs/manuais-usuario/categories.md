# Organização de Categorias 🏷️

As **Categorias** são essenciais para classificar suas receitas e despesas, permitindo relatórios precisos e insights sobre seus hábitos financeiros.

## 🌳 Hierarquia: Pais e Filhos

O sistema utiliza uma estrutura em dois níveis para melhor organização:

1.  **Categorias Pai (Raiz):** Grupos principais (ex: "Moradia", "Transporte").
2.  **Subcategorias:** Detalhamentos específicos (ex: dentro de "Moradia" você pode ter "Aluguel", "Luz", "Internet").

## 🛠️ Campos da Categoria

### Identidade

- **Tipo (Receita/Despesa):** Define se a categoria representa dinheiro entrando ou saindo.
- **Nome:** Título curto e claro (ex: "Salário", "Supermercado").
- **Categoria Pai:** Se selecionada, esta categoria se tornará uma subcategoria. Se deixada em branco, será uma categoria principal.

### Aparência Visual

- **Ícone:** Identificador visual usado em toda a plataforma (listas, gráficos, dashboards).
- **Cor Principal:** Usada em gráficos de pizza e barras para diferenciar os setores de gasto.

## 💡 Dicas de Organização

- **Simplifique:** Não crie centenas de categorias. Mantenha grupos lógicos que façam sentido para sua análise mensal.
- **Padrão de Cores:** Use cores quentes (vermelho, laranja) para gastos e cores frias (verde, azul) para receitas.
- **Subcategorias:** Use-as para despesas que você quer monitorar de perto (ex: em vez de apenas "Saúde", use subcategorias para "Farmácia" e "Consultas").
